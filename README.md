# 🩺 VEDAZ - Expert Booking System

> A MERN Stack application to find and book expert consultations

## 🚀 Tech Stack

- **Frontend:** React.js + Vite
- **Backend:** Node.js + Express.js  
- **Database:** MongoDB
- **Styling:** CSS3

## ✨ Features Implemented - MVP v1.0

1. **Expert Listing Screen**
   - Displays expert cards with Name, Expertise, Price, Rating, Experience
   - Loading & Error states handled
   - Clean dark theme UI

2. **Booking System**
   - "Book Now" button opens detail modal
   - Shows expert details + available time slots
   - Booking confirmation with success alert

3. **API Integration**
   - REST API built with Express.js
   - GET `/api/experts` endpoint
   - MongoDB integration for expert data
