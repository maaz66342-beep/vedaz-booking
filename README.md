# 🤖 VEDAZ v2.0 - Voice Assistant 

> **Note:** Repo name is `vedaz-booking` but this is VEDAZ Voice Assistant project.

Voice Enabled Desktop Assistant built with Node.js + Express. Control your PC using Hinglish voice commands.

## ⚡ v2.0 Major Update - Tech Stack Migration

## 🚀 Setup & Run
1. Clone: `git clone https://github.com/maaz66342-beep/vedaz-booking.git`
2. Install: `npm install`
3. Start: `npm start` 
4. Open: `http://localhost:5000`
